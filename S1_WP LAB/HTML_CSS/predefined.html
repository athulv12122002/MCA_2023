String and Math Objects in JavaScript

String Object

The String object provides methods for working with strings:

Example 1: Concatenation

        
            var str1 = "Hello";
            var str2 = "World";
            var result = str1.concat(" ", str2);
            console.log(result); // Output: Hello World

Example 2: Length

        
            var str = "JavaScript";
            var length = str.length;
            console.log(length); // Output: 10

Example 3: Substring

        
            var str = "Hello World";
            var substring = str.substring(6, 11);
            console.log(substring); // Output: World

Math Object

The Math object provides mathematical constants and functions:

Example 4: Random Number

        
            var randomNumber = Math.random();
            console.log(randomNumber);

Example 5: Round a Number

        
            var num = 5.67;
            var roundedNum = Math.round(num);
            console.log(roundedNum); // Output: 6

Example 6: Square Root

        
            var squareRoot = Math.sqrt(25);
            console.log(squareRoot); // Output: 5
